(inner) resizeImage(source, settings) → {Promise.<*>}

Load and resize an image to specified format. Safari does not support `options.resize*` in `createImageBitmap()`; therefore we must draw the bitmap to canvas, instead of resizing it by calling `creatImageBitmap()`.
